Triangle Problem app help

Hello! This is help file for Triangle Problem app.

Purpose

With this application you can determine type of your triangle. Possible types are:
  • scalene;
  • isosceles;
  • equilateral.

How to use

To get triangle type, just enter its sides lengths in fields that are described as "Side #.." and press "Analyze" button. After that, you will get message with type of triangle if analyze was succeed, otherwise message will contain analyze fault reason.

Restrictions

Restrictions on input data are:
  • values are positive integers;
  • any value shouldn't exceed 264 - 1;
  • sum of any two sides lengths shouldn't exceed 264 - 1.
